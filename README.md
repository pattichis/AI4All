The repository provides a collection of lessons for teaching AI to students who have completed 9th grade.

# Course goals
- Foundational concepts of Neural Networks and modern AI.
- Introduce Python, NumPy, and PyTorch.
- Introduce Python for understanding digital images.
- Introduce Python for building, training, and testing AI systems.
- Introduce Generative AI concepts.
